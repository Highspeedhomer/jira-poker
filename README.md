# 🃏 Jira Poker

Real-time planning poker for your team — built with vanilla JS + Firebase Realtime Database.

**Live site:** `https://YOUR-USERNAME.github.io/jira-poker`

## Features

- 🔴 Live multiplayer — votes sync instantly across all devices
- 🔒 Private votes — hidden until the host reveals
- ⚡ Speed round mode — no ticket needed, instant reset
- 📊 History & analyst trends — persisted in Firebase
- ↩ Rejoin — remembers your last session on reopen
- 👑 Host controls — reveal, end session, vote on behalf of absent members

## Deploy (GitHub Pages)

See [SETUP.md](SETUP.md) for full step-by-step instructions.

## Update the app

After making changes to `index.html`:

```bash
git add index.html
git commit -m "Update app"
git push
```

GitHub Pages redeploys automatically in ~30 seconds.
